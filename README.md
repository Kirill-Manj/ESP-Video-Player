# ESP-Video-Player
This is a display of 64x128 for any .mp4 files that are ~1 minute or less.
Requires ESP32 (DEVKIT V1 for the board selection). 1.3' OLED IIC.
I used AI for this when I needed assistance and to optamize my code.

Tutorial video here: https://www.instagram.com/p/DaEul_mv1i0/
